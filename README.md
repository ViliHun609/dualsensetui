# dualsensetui
 A REALLY simple Linux TUI for dualsensectl with omarchy implementations.

(it's kinda vibe coded but i think it still can be useful for other so i publish it.)

## Things that dualsensetui can do

- Turn off controller ✅
- Display battery level of your controller ❌
